# E-Commerce Customer Analysis

Data analysis project using Python, Pandas, Matplotlib, and Seaborn to explore customer behavior, revenue trends, churn rate, and product category performance in an e-commerce dataset.

## Tools
* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset
* ecommerce_customer_data_large.csv

## Analysis
* Data Cleaning
* Exploratory Data Analysis (EDA)
* KPI Analysis
* Customer Churn Analysis
* Revenue Trend Visualization
